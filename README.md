# miunlock
### Installation
- apt update
- apt install git
```bash
git clone https://github.com/R1punk/miunlock
```
- cd miunlock
- chmod +x setup.sh un.sh
- dpkg -i adb-fastboot_0.2.0-5_aarch64.deb
- apt install -f
- ./setup.sh
- ./un.sh
